数据类型

分类

  • 基本类型
    • string：任意字符串
    • number：任意数字
    • Boolean：true/false
    • undefined：undefined，未赋值的var变量
    • null：null
  • 对象（引用）类型
    • object：任意对象
    • function：特别的对象（可以执行）
    • array：特别的对象（数值下标，内部数据有序）

类型判断

  • typeof：

    特殊情况记一下
    • 可以判断: undefined/ 数值 / 字符串 / 布尔值 / function
    • 不能判断: null与object object与array

  • instanceof：

    • 可以判断对象类型

  • ===:

    • 可以判断undefined和null

问题

  • undefined 和null的区别
    • undefine:代表变量定义未赋值
    • null：变量的一种特殊的值
  • 何时使用null？
    • 初始赋值为null，在开始对象内容不明确时，表明将来赋值为对象
    • 结束时赋值为null，内存回收，被垃圾回收器回收
  • 严格区分变量类型和数据类型
    • 数据的类型
      • 基本类型
      • 对象类型
    • 变量的类型(变量内存值的类型)
      • 基本类型: 保存就是基本类型的数据
      • 引用类型: 保存的是地址值

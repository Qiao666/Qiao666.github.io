js高级-函数

函数（基础）

什么是函数

  • 实现特定功能的多条语句的封装体

  • 只有函数可以执行 其他类型数据无法执行

    为什么要用函数

  • 提高代码复用

  • 便于阅读和交流

    如何定义

  • 函数声明

    function fn1 () { //函数声明
    console.log('fn1()')
}

  • 表达式

    var fn2 = function () { //表达式
    console.log('fn2()')
}

二者有区别/

如何调用

  • test(): 直接调用
  • obj.test(): 通过对象调用
  • new test(): new调用
  • test.call/apply(obj): 临时让test成为obj的方法进行调用

回调函数

  • 特殊的函数

    • 用户或者系统内部定义的
    • 没有调用
    • 最终函数执行了

  • 常见的回调函数

    • dom事件回调函数

      document.getElementById('btn').onclick = function () { // dom事件回调函数
alert(this.innerHTML)
}

    • 定时器回调函数

      setTimeout(function () { // 定时器回调函数

alert('到点了'+this)
}, 2000)
